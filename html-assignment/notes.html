Notes

  • < p > < / p >. This is the paragraph tag. This adds a paragraph to your web page.
  • < body > < / body >. This is the body tag. This tag is used to help organize your code. everything that is inside this tag is the body of your web page.
  • < ul > < / u l >. This is the unordered list tag. This tag allows you to make a list without a numbered order.
  • < header > < / header >. This is the header tag. This allows you to make a header at the top of your web page.
  • < footer > < / footer >. This is a footer tag. This allows you to add a footing to the bottom of your page.
